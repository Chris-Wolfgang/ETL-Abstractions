Methods

DisposingOwned(IEtlPipelineSink, params object[])

Wraps a sink so that the given resources — opened by the code that built the pipeline (for example a file stream a path-based loader factory created) — are disposed after the run completes, whether it succeeds or throws.

public static IEtlPipelineSink DisposingOwned(this IEtlPipelineSink sink, params object[] ownedResources)

Parameters

sink IEtlPipelineSink

The sink to wrap.

ownedResources object[]

The resources to dispose after the run. Each is disposed via IAsyncDisposable when supported, otherwise IDisposable; anything else (or null) is skipped. Resources are disposed in reverse of the order supplied (LIFO).

Returns

IEtlPipelineSink

A sink that disposes ownedResources after running, or the original sink unchanged when no resources are supplied.

Exceptions

ArgumentNullException

sink is null.
